مهارت هایی که در این شرکت اموختم

  1. ساختارها و الگوریتم‌های داده: تسلط به مفاهیم ساختارها و الگوریتم‌های داده برای حل مسائل پیچیده.
  2. پایگاه داده و SQL: آشنایی با پایگاه‌های داده و زبان SQL برای ذخیره و استخراج اطلاعات.
  3. زبان‌های برنامه‌نویسی شی‌گرا (OOP): تسلط به مفاهیم شی‌گرایی و نوشتن کد قابل توسعه.
  4. محیط‌های توسعه یکپارچه (IDE): استفاده از ابزارهای توسعه کد مانند Visual Studio Code یا PyCharm.
  5. رایانش ابری: آشنایی با مفاهیم رایانش ابری و استفاده از سرویس‌های مانند AWS یا Azure.
  6. توسعه وب: تسلط به تکنولوژی‌های وب مانند HTML، CSS، JavaScript و فریم‌ورک‌های مختلف.
  7. کانتینرها: آشنایی با Docker و Kubernetes برای مدیریت برنامه‌ها.
  8. ویرایشگرهای متن: تسلط به ویرایشگرهای متن مانند Visual Studio Code یا Sublime Text.
  9. کنترل نسخه Git: آشنایی با Git برای مدیریت کدها و همکاری در تیم‌ها.
